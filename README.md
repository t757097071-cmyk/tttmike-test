# 锦愿阁｜学业祈福

一个中文古风祈福网站，使用 React + TypeScript + Vite 开发。当前内测版包含本地/轻量 Node API，用于手机号登录、愿力钱包、测试额度订单、点灯记录和后台统计。

## 运行

```bash
pnpm install
pnpm run api
pnpm run dev
```

生产模式：

```bash
pnpm run build
pnpm run start
```

部署说明见 [DEPLOYMENT.md](./DEPLOYMENT.md)。

## 木鱼音效

项目已写好音频播放逻辑。若需要真实木鱼声，请把音频文件放到：

```text
src/assets/woodfish.mp3
```

如果没有该文件，页面仍可正常运行，点击木鱼时会在控制台提示音频文件缺失。

## 功能

- 木鱼点击祈福：今日功德 +1、连续祈福天数计算、浮动动画；总祈福次数预留给收费祈福服务，不随木鱼点击变化。
- 祈福分类：学业、事业、健康、平安。
- 写下今日心愿：最多 50 字，保存最近祈愿记录。
- 点亮心愿笺：每日首份免费，当天继续加写每份消耗 66 愿力，愿力不足时可前往领取测试额度。
- 点亮心愿灯：高阶点灯服务，单独保存心愿灯记录。
- 领取愿力：内测免费领取测试额度，不接真实支付，保存测试额度记录。
- 我的页面：查看愿力、功德、心愿记录、点灯记录和测试额度记录。
- 今日签文：每次刷新随机展示。
- 抽签：每日首签免费，当天继续加抽每签消耗 66 愿力。
- 今日黄历：按日期生成黄历宜忌、四象建议、吉位、时辰、学习计划与分享卡片。
- 祈福排行榜：前端模拟数据结合本地数据展示。
- 我的祈愿记录：本地保存，可二次确认后清空。

## 页面

- `/` 首页
- `/blessing` 祈福
- `/lamps` 心愿灯
- `/draw` 抽签
- `/huangli` 今日黄历
- `/recharge` 领取愿力测试额度
- `/profile` 我的锦愿阁

## 合规提示

本页面内容仅作传统文化体验与情绪陪伴使用，不承诺现实结果，不替代医疗、法律、投资、升学等专业建议。愿力值为站内虚拟权益，仅用于锦愿阁相关功能体验。
